Particle/Nuclear Physics

I worked for Roosevelt University and Dr. Brent Barker on the modification of particle collision code written in FORTRAN by Dr. Pawel Danielewicz. Dr. Danielewicz is a professor at Michigan State University and works in their physics department. Brent and I worked and modified the code so that it could be run using the linux fortran compiler, gfortran, as well as that intel fortran compiler, ifort. I also was able to get the code working using gfortran in Windows Powershell. We were also able to look at the isotope yields of colliding particles to analyze the output. I will put all of our data up on this website so that anyone can take a look once I get some time.

Roosevelt University Drug Consortium

In one of my math classes, we were sent on a mission to work with a mentor at Roosevelt University and work on a project involving statistical analysis and data mining. The project that I worked on had to deal with the rise in people being admitted into federal heroin treatment facilities. We had to find correlations within the patients information that could point to possible reasons for the increase. We worked primarily with SPSS, but I did more of the data mining with R.
